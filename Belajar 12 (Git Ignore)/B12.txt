Git Ignore = untuk membuat file tertentu tidak masuk ke dalam git add .

nama file
.gitignore

langkahnya. buat file dengan nama ".gitignore"
lalu masukan file apa saja yang tidak akan di masukan contoh "config.php"

		data/ = tidak akan memasukan semua data yang ada pada folder data/
		*.exe = file semua yang .exe tidak akan terbawa
